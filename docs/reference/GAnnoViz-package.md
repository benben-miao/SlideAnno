# GAnnoViz: Genomic Annotation and Visualization

The GAnnoViz package provides tools for genomic annotation using sliding
windows and GFF/GTF parsing.

## Main Functions

- [`extract_genes`](https://github.com/benben-miao/GAnnoViz/reference/extract_genes.md):
  Extract gene annotations from GFF/GTF.

- [`extract_exons`](https://github.com/benben-miao/GAnnoViz/reference/extract_exons.md):
  Extract exon regions.

## Utilities

- [`extract_mrnas`](https://github.com/benben-miao/GAnnoViz/reference/extract_mrnas.md):
  Extract mRNA transcripts.

- [`extract_cds`](https://github.com/benben-miao/GAnnoViz/reference/extract_cds.md):
  Extract coding sequences (CDS).
