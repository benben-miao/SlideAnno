# Package index

## Shinyapp

- [`GAnnoVizApp()`](https://github.com/benben-miao/GAnnoViz/reference/GAnnoVizApp.md)
  : Launch GAnnoViz Shiny application

## Extract Features

- [`extract_promoters()`](https://github.com/benben-miao/GAnnoViz/reference/extract_promoters.md)
  : Extract promoter ranges from GFF or GTF
- [`extract_utr5()`](https://github.com/benben-miao/GAnnoViz/reference/extract_utr5.md)
  : Extract 5'UTR ranges from GFF or GTF
- [`extract_genes()`](https://github.com/benben-miao/GAnnoViz/reference/extract_genes.md)
  : Extract genes information from GFF or GTF
- [`extract_mrnas()`](https://github.com/benben-miao/GAnnoViz/reference/extract_mrnas.md)
  : Extract mRNA transcripts from GFF or GTF
- [`extract_cds()`](https://github.com/benben-miao/GAnnoViz/reference/extract_cds.md)
  : Extract CDS ranges from GFF or GTF
- [`extract_exons()`](https://github.com/benben-miao/GAnnoViz/reference/extract_exons.md)
  : Extract Exons ranges from GFF or GTF
- [`extract_utr3()`](https://github.com/benben-miao/GAnnoViz/reference/extract_utr3.md)
  : Extract 3'UTR ranges from GFF or GTF

## Plot Structure

- [`plot_gene_stats()`](https://github.com/benben-miao/GAnnoViz/reference/plot_gene_stats.md)
  : Plot gene stats for chromosomes
- [`plot_gene_structure()`](https://github.com/benben-miao/GAnnoViz/reference/plot_gene_structure.md)
  : Plot gene structure (Promoter, 3'UTR, Exon, Intron, 5'UTR)
- [`plot_gene_domains()`](https://github.com/benben-miao/GAnnoViz/reference/plot_gene_domains.md)
  : Plot protein domains from Ensembl
- [`plot_interval_structure()`](https://github.com/benben-miao/GAnnoViz/reference/plot_interval_structure.md)
  : Plot gene structures for a genomic interval
- [`plot_interval_flank()`](https://github.com/benben-miao/GAnnoViz/reference/plot_interval_flank.md)
  : Gene neighborhood architecture around a focal gene
- [`plot_chrom_structure()`](https://github.com/benben-miao/GAnnoViz/reference/plot_chrom_structure.md)
  : Plot chromosome structures and gene stats
- [`plot_chrom_genes()`](https://github.com/benben-miao/GAnnoViz/reference/plot_chrom_genes.md)
  : Plot chromosome structures and gene annotation
- [`plot_chrom_heatmap()`](https://github.com/benben-miao/GAnnoViz/reference/plot_chrom_heatmap.md)
  : Plot genomic feature density heatmap

## DEG Anno & Viz

- [`anno_deg_chrom()`](https://github.com/benben-miao/GAnnoViz/reference/anno_deg_chrom.md)
  : Annotate differentially expressed genes (DEGs) with chromosome
  positions
- [`plot_deg_chrom()`](https://github.com/benben-miao/GAnnoViz/reference/plot_deg_chrom.md)
  : Plot differentially expressed genes (DEGs) hyper/hypo distributions
  by chromosome
- [`plot_deg_exp()`](https://github.com/benben-miao/GAnnoViz/reference/plot_deg_exp.md)
  : Plot DEGs up/down along chromosomes

## SNP Anno & Plot

- [`anno_fst_dmr()`](https://github.com/benben-miao/GAnnoViz/reference/anno_fst_dmr.md)
  : Annotate FST/DMR slide windows with genomic features
- [`plot_snp_density()`](https://github.com/benben-miao/GAnnoViz/reference/plot_snp_density.md)
  : Plot SNP density at chromosome level
- [`plot_snp_fst()`](https://github.com/benben-miao/GAnnoViz/reference/plot_snp_fst.md)
  : Plot genomic weighted FST heatmap
- [`plot_snp_anno()`](https://github.com/benben-miao/GAnnoViz/reference/plot_snp_anno.md)
  : Plot genomic FST with Top-N gene annotations

## DMG Anno & Plot

- [`anno_fst_dmr()`](https://github.com/benben-miao/GAnnoViz/reference/anno_fst_dmr.md)
  : Annotate FST/DMR slide windows with genomic features
- [`plot_dmg_chrom()`](https://github.com/benben-miao/GAnnoViz/reference/plot_dmg_chrom.md)
  : Plot differentially methylated regions (DMRs) hyper/hypo
  distributions by chromosome
- [`plot_dmg_exp()`](https://github.com/benben-miao/GAnnoViz/reference/plot_dmg_exp.md)
  : Plot DMGs hyper/hypo along chromosomes
- [`plot_dmg_trend()`](https://github.com/benben-miao/GAnnoViz/reference/plot_dmg_trend.md)
  : Plot chromosomal DMGs trend
